# Car Sales Advertisement Dashboard

This project is a Streamlit dashboard for exploring a dataset of used vehicle advertisements.
It provides a preview of the dataset and interactive visualizations to analyze price patterns
and the relationship between mileage and price.

## Features
- Dataset preview table
- Checkbox filter to show only vehicles above $20,000
- Histogram: price distribution
- Scatter plot: odometer vs. price (colored by fuel type)

## Tech stack
- Python
- pandas
- Streamlit
- Plotly Express
